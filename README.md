# NBC News Scrapy Crawler

A web scraping project created using **Scrapy** that extracts the latest news articles from **NBC News**.  
The spider collects article **titles, links, publication dates, and summaries**.

## Features
- Crawls the **NBC News homepage**
- Extracts **headline titles and article links**
- Visits each article page to collect:
  - Title
  - Link
  - Source
  - Publication date
  - Article summary
